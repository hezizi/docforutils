# 开始使用

## 安装

::code-group

  ```bash [npm]
  npm install @yuci/utils
  ```

  ```bash [yarn]
  yarn add @yuci/utils
  ```

  ```bash [pnpm]
  pnpm add @yuci/utils
  ```

::

```bash
import { debounce, throttle, capitalizeFirstLetter, ... } from '@yuci/utils'
```
